## 🚀 立即配置（3步骤，5分钟）

### 步骤1：申请百度地图AK（2分钟）

1. **访问**：https://lbsyun.baidu.com/apiconsole/key/create
2. **登录**百度账号（没有就注册一个）
3. **创建应用**：
   - 应用名称：`高铁巡逻系统`
   - 应用类型：`浏览器端`
   - Referer白名单：`*`
4. **复制AK**：创建成功后显示的密钥

### 步骤2：配置AK（1分钟）

打开 `index.html` 文件，找到第10行：

```html
<script type="text/javascript" src="https://api.map.baidu.com/api?v=3.0&ak=YOUR_BAIDU_MAP_AK"></script>
```

将 `YOUR_BAIDU_MAP_AK` 替换为您的AK：

```html
<script type="text/javascript" src="https://api.map.baidu.com/api?v=3.0&ak=您刚才复制的密钥"></script>
```

**示例**：
```html
<script type="text/javascript" src="https://api.map.baidu.com/api?v=3.0&ak=abcdefghijklmnopqrstuvwxyz123456"></script>
```

### 方案C：终极方案
如果百度地图也不行（偏移仍然很大），可能是：
1. **手机GPS硬件问题**（某些手机GPS芯片精度有限）
2. **当地GPS信号问题**（高楼密集区）

这种情况下，我们可以：
- 尝试腾讯地图（也是GCJ-02）
- 使用手机APP的定位数据（通过二维码/API传输）
- 购买专业GPS设备

## 💡 技术解释

### 为什么换百度？

```
高德地图定位
├─ 坐标系：GCJ-02
├─ 定位：GPS + WiFi + 基站
└─ 您的结果：❌ 偏移近1公里

百度地图定位
├─ 坐标系：BD-09（独有）
├─ 定位：GPS + WiFi + 基站 + POI
└─ 预期：✅ 10-50米精度
```

### 坐标系对比

| 坐标系 | 使用者 | 特点 |
|--------|--------|------|
| WGS-84 | GPS卫星 | 真实坐标 |
| GCJ-02 | 高德/腾讯 | 加密坐标 |
| BD-09 | 百度 | 二次加密 |
